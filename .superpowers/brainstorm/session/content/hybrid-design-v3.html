🔄 Логика компонента «Скорость»

Аудио (BPM-режим)

  • Центральное значение — BPM трека
  • Бегунок: диапазон от BPM−30 до BPM+30
  • Кнопки ± по краям: шаг ±1 BPM
  • Отображается по умолчанию, пока не обнаружено видео

Видео (множитель)

  • Автопереключение при обнаружении video-элемента
  • Бегунок: диапазон 0.25x – 2x
  • Те же кнопки ± по краям: шаг ±0.05x
